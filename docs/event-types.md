# Types d'événements

## Événements gérés

| Événement | Source | Notification |
|-----------|--------|-------------|
| Nouveau message | messaging-service | Push + badge |
| Demande de contact | user-service | Push |
| Média uploadé | media-service | Push |
| Contenu modéré | moderation-service | Push (si rejeté) |
| Message programmé envoyé | scheduling-service | Push |
